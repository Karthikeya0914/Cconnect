# 🏦 CConnect — Bank Officer Daily Tracker

A full stack web application built for bank officers to manage 
customer calls, daily tasks and pending works efficiently.

## 🚀 Live Demo
[Click here to view the app](https://your-netlify-url.netlify.app)

## 💡 Problem It Solves
Bank officers contact multiple customers daily but struggle to:
- Remember remarks from each call
- Track pending follow-ups
- Manage daily tasks in one place

CConnect solves all of this in one simple dashboard.

## ✨ Features
- 🔐 Secure login & registration
- 📞 Log customer calls with account number, contact, purpose & remarks
- ✅ Daily task management with pending/completed tracking
- ⏰ Pending works view — all pending calls & tasks in one place
- 🔍 Search customers by name or account number

## 🛠️ Built With
- React + TypeScript (Frontend)
- Firebase Firestore (Database)
- Firebase Authentication (Login system)
- Vite (Build tool)
- Netlify (Deployment)
